Cruor Open-Source Statement
----
----

We believe that while the foundational implementation of our game systems are core to the experience, they are not fundamentally our product.  Therefore we intend to openly share said implementations to be used by and or expanded by anyone who finds them useful.

Modifications to the original Lyra Code will be surrounded by comments starting with "@CruorRTS_CodeModification" including a brief description of the purpose of the modification.

Code from other projects will be surrounded by comments starting with "@CruorRTS_CodeAttribution" including who the code is attributed to.

Lyra Content Information
----
----

When downloading this source code from git, the Lyra content folders are not included.

To make full use of the the contained source code, you will need to download the content from the Unreal Marketplace through the Epic Games Launcher.

For more information please visit the [Unreal Engine Lyra documentation](https://docs.unrealengine.com/5.0/en-US/lyra-sample-game-in-unreal-engine/)

Once installed, copy the "Content" folder and any other folders named "Content" from the Marketplace project you downloaded to the project in your solution, and you should be good to go.
